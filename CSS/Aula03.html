O que é o box model: è uma caixa retangular, essa caixa possui propriedades de uma caixa 2D. Ele é fundamental para fazer layouts para a web, maior facilidade de aplicar o css. Cada elemento da página será considerado uma caixa...

As propriedades são:
  • Tamanho (largura x altura) = width and height. O tamanho total da caixa será feito pelo content-box.
  • Conteúdo = content
  • Bordas = border
  • Preenchimento interno = padding
  • Espaços fora da caixa = margin
Display: block vs inline
  • Como as caixas se comportam em relação as outras caixas
  • Comportamento externo das caixas
Comportamento do Display block:
  • Ocupa toda a linha, colocando o próximo elemento abaixo desse
  • width e height são respeitados
  • padding, margin, border irão funcionar normalmente
  • "p" "div" "section" todos os headings "h1" "h2"...
Comportamento do Display inline
  • Os elementos ficam ao lado do outro e não empurram outros elementos para baixo.
  • width e height não funcionam.
  • Somente valores horizontais de margin.
  • "a"" "strong" "span" " em"
Margin

È o espaço (margem) entre os elementos
Podemos dividir o margin em 4 valores:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left
  • values: length, percentage \ auto
Shorthand da margin. *cuidado com margin collapsing, top se ajunta com bottom*
  • margin: 12px"top" , 16px"right" , 10px"bottom" , 4px"left";
  • margin: 12px"top" , 16px "laterais", 0"bottom";
  • margin: 8px"top e bottom", 16px"laterais";
  • margin: 8px "aplica para todos as dimensoes"
Por padrão, o navegador Chrome já aplica 8px de margin, isso está na documenta~ ao.
Padding

Meu padding faz meu conteúdo se afastar da borda da minha box. o Paddin nao faz as caixas serem empurradas, ela é o espaço interno.

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

O padding pode ter valores (values) de comprimento (px, em, rem) ou de porcentagem (%).Na aula sobre box-xizing aprendemos como resolver essa diferença na largura do elemento

Outline

O outline é muito semelhante ao border, mas difere em 4 sentidos:

  • Não modifica o tamanho da caixa, pois não é parte do Box Model
  • Poderá ser diferente de retangular
  • Não permite ajuste individuais
  • Mais usado pelo user agent para acessibilidade, no caso de um input
