O que é o box model: è uma caixa retangular, essa caixa possui propriedades de uma caixa 2D. Ele é fundamental para fazer layouts para a web, maior facilidade de aplicar o css. Cada elemento da página será considerado uma caixa...

As propriedades são:
  • Tamanho (largura x altura) = width and height
  • Conteúdo = content
  • Bordas = border
  • Preenchimento interno = padding
  • Espaços fora da caixa = margin
